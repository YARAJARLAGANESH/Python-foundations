Python Foundations – Week 1
Overview

This repository contains my Week 1 practice work while building a strong foundation in Python programming.
The focus was on core concepts, writing clean functions, and understanding how to handle data and errors properly.

Topics Covered
1. Data Types

Worked with fundamental Python data types:

int, float
str
list, tuple, set, dict
bool

Implemented examples to understand:

Mutability vs immutability
Type conversions
Basic operations on each type
2. Functions

Practiced function design using:

def and return
Default arguments
*args (variable positional arguments)
**kwargs (variable keyword arguments)
Implemented:
10 functions using combinations of *args and **kwargs
3. String Manipulation

Explored commonly used string methods:

split(), join()
strip(), replace()
format(), f-strings
Project:
Built a string manipulation script using 8+ string methods
Focused on chaining operations and handling edge cases
4. List & Dictionary Comprehensions

Replaced traditional loops with:

List comprehensions
Dictionary comprehensions
Implemented:
5 optimized examples converting for-loop logic into comprehensions
5. Exception Handling

Learned how to write robust code using:

try, except, finally
raise for custom errors
Project:
Built a mini calculator
Handles invalid input
Prevents crashes (e.g., division by zero)
Uses structured exception handling
